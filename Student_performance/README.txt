Student Management System

Description:

This is a simple command-line Student Management System built using
Python and Pandas. It allows you to add, update, delete, search
students, and generate reports.

Requirements:

-   Python 3.14.3
-   Pandas library

To install Pandas, run: pip install pandas

Required Files:

-   Student_Performance_System.py (Python file)
-   students.csv (data file)

If students.csv does not exist, create it with the following headers:
id,name,math,english,science,history

How to Run:

Open a terminal in the project directory and run: python
Student_Performance_System.py


Features:

1.  Add Student
2.  Update Student Name
3.  Update Student Marks
4.  Delete Student
5.  Search Student
6.  Generate Student Report
7.  Find Top Student (by average score)
8.  Find Top Student (by subject)
9.  Exit

Generated Files:

-   students.csv (Stores all student records)
-   StudentReport.csv (Generated report with average and grade)

Exit:

Choose option 9 to exit the system safely.
